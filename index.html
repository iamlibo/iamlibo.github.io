在肯德基观察

发表于 2021-11-21

前几天记录了一下在肯德基学习，考完试后就相对轻松一些。可以看一下其他的书籍，在看书的同时也观察也一下周围。

每次来肯德基读书学习，我基本都呆一上午的时间，每次都会遇到除了营业员之外的一些人，他们也是长期的在这里『蹭』地方的人，形形色色。

  • 一位疑似精神有些问题的人，拿着两个大旅行袋，身上的衣服比较脏乱，脸上也是脏兮兮的。就是静静地在一个位置上坐着，非常怕人注视她，只要有人注视她就会用手把脸遮起来。我早上7点多来，她就已经在某个偏僻的角落坐着了，我11点多走的时候，她还是在那里坐着，多数如此。
  • 一个流浪汉，身材高大，穿着破乱，冬夏都一样的衣服。这个人进来后就直奔垃圾桶，翻着吃的拿出来就吃，嘴里还哼着只有自己才能听懂的小曲，脸上带着满意的笑容。吃完就走，也许还要到其他的地方继续翻找能吃的东西。

这两个人都是普通人眼中的异类，都是社会的最底层，只有生存的本能，但不是靠劳动生存，而是靠拾荒或者其他的方式生存。他们的精神世界是什么样的呢？我现在无法得知，但肯定是普通的大多数人是不同的，普通人的是靠劳动挣钱，换取生存的资源。今天正好看一本书，彭凯平《孩子的品格》，副标题是：写给父母的积极心理学。我想到如果这两个人的父母或家人在他们出现问题的时候给予他们精神层面的积极帮助， 现在的社会物质是非常丰富的，只要肯付出来劳动就能够获得生存资源，不至于翻垃圾桶。

  • 第三位是一个老太太，是80后，驼背。大约在9点左右来，一般都拉一个买菜的购物车，坐在离门很近的一个位置，然后开始玩手机，用微信语言聊天。大致的意思是好象通过手机在做一些什么活动，类似看视频返利、点赞什么的，偶尔还要拉几个老姐妹一块，教她们操作。
  • 第四位是一位披衣男，在秋天的时候就是披着一件棉衣，拿一个布袋子，坐在那玩两部手机，看视频很大声，比我去得要早，就是一直在刷各种小视频。
  • 第五位是一个睡觉男，也是比我去的还要早，一个黑色的双肩包，一个红色的手提袋。大部分时间都是在睡觉，

我是每个周末两天去肯德基，遇到的时间只是2/7可能，并不能智能他们的全部生活，也许只是和我一样在周围办事，临时蹭个地方呆一会，也许他们眼中的我也是类似的样子。生活真的很奇妙！

Kong 网关入门

发表于 2019-06-19 更新于 2021-11-21

Kong介绍

Kong是一款基于Nginx_Lua模块写的高可用网关API，通过前置的负载均衡配置把请求均匀地分发到各个Server，来应对大批量的网络请求。基于Nginx 特性，Kong本身也非常容易地扩展到多个服务器上。

Kong主要有三个组件：

Kong Server ：基于nginx的服务器，用来接收API请求。
Apache Cassandra/PostgreSQL ：用来存储操作数据。
Kong dashboard：官方推荐UI管理工具，也可以使用 restfull 方式 管理admin api。
